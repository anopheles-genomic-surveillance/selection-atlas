Data sources#

This report analyses data from the Malaria Vector Genome Observatory. The current analysis version is agam/2025.03.05 which includes data for a total of 4,878 mosquitoes sampled from 25 countries. Please see the table below for more information about the sample sets and releases from which the data were obtained.
